# Working dependency set for the shared .venv (Python 3.12, Windows).
# Captured from `pip freeze` 2026-07-03. There was previously no requirements
# file at all (pyproject.toml declares zero runtime deps) - this is a curated
# reference snapshot, not a from-scratch installer: torch's CUDA build in
# particular needs a non-default index and can't be expressed as a plain pin.
#
# Install order that actually worked this session:
#   pip install opencv-python fer deepface hsemotion-onnx mediapipe scikit-learn tf-keras
#   pip install --index-url https://download.pytorch.org/whl/cu124 torch torchvision
#   pip install transformers==4.49.0 accelerate bitsandbytes qwen-vl-utils timm einops pillow
#   pip install pyvips emotiefflib
# Plus a native (non-pip) dependency: libvips DLL for pyvips, fetched into
# data/vips/ - see src/vlm/live_webcam_vlm_demo.py header for the exact path
# and download source (github.com/libvips/build-win64-mxe releases).
